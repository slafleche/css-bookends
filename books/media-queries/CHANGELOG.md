# @css-bookends/media-queries

## 0.1.0

### Minor Changes

- First CSS-Bookends beta.

  - `css-calipers` is now purely the measurement layer. The media query helper has
    been removed and moved to its own package, `@css-bookends/media-queries`. Users
    who need media queries can switch to CSS-Bookends or stay on `css-calipers@0.14`.
  - `@css-bookends/media-queries` is the first release of the media query helper as
    a standalone book, built on `css-calipers`.

### Patch Changes

- Updated dependencies
  - css-calipers@0.15.0
