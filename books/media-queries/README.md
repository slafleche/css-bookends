# @css-bookends/media-queries (removed)

This book has been **removed from the active workspace**. It is no longer a
pnpm package here: its `src/`, `tests/`, build config, and `package.json` are
gone, so nothing builds, tests, or publishes from this directory anymore.

The code is not lost. It still exists in this repo's **git history**, and the
last published versions remain on npm under `@css-bookends/media-queries`. If
you need the old implementation, check out an earlier commit or install a prior
published version.

It was pulled because it needs **serious rework before it returns**. The design
notes and the missing-feature backlog for that rework are kept alongside this
tombstone:

- [`improvements.md`](./improvements.md): the rework backlog (missing media
  features, logical operators, and other gaps).
- [`media-queries-space.md`](./media-queries-space.md): the full CSS media
  query value surface (the spec-level reference the reworked book should cover).

When the book is rebuilt, it will return as an active workspace package.
