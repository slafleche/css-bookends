## Missing Media Query Features (Examples)

### Device width, height, aspect ratio

```css
// Example intent:
// @media screen and (device-width: 1200px)
// @media screen and (device-height: 800px)
// @media screen and (device-aspect-ratio: 16/9)
```

### Color, color index, monochrome

```css
// Example intent:
// @media screen and (color: 8)
// @media screen and (color-index: 256)
// @media screen and (monochrome: 1)
```

### Grid

```css
// Example intent:
// @media screen and (grid: 1)
```

### Scan

```css
// Example intent:
// @media tv and (scan: interlace)
```

### Light level

```css
// Example intent:
// @media screen and (light-level: dim)
```

### Prefers reduced transparency

```css
// Example intent:
// @media screen and (prefers-reduced-transparency: reduce)
```
